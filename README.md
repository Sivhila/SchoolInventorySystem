
# School Inventory Management System

A Flask-based web application for managing school inventory items, issuing equipment, tracking returns, overdue items, and user roles.


## Features

- User authentication (login, register, logout)
- Password hashing for security
- Role-based access (admin & staff)
- Item management
- Issue & return system
- Due date tracking
- Overdue monitoring
- Activity logging
- Low stock monitoring


## Tech Stack

- Python (Flask)
- SQLite3
- Bootstrap 5
- HTML, CSS, JavaScript
